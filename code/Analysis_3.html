Main Findings

  1. The best model to describe PIPO seedling growth with the data I have is
M6F.pipo <- gls(LastYearGrth.cm ~ Ht.cm +  BasDia.cm + Ht1.2 + Fire + Fire*Ht.cm,
                weights = varPower,
                 data= dfPIPO, method = "ML")
  1. DIFN is not useful for PIPO
  2. Shrub cover is not important but shrub height is
  3. Shrub genus does not appear to be important
  4. The effect of shrub height on seedling growth is small compared to the effect of seedling height and fire
